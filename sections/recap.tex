\begin{frame}{Recap}
\textbg{ROS 2} software is organized in \textbg{packages}, built by \textbg{\texttt{colcon}} calling either \textbg{CMake} or \textbg{setuptools}.
\newline\newline
\textbg{Messages} are the most basic communication paradigm, entirely built on DDS layer communication APIs.
\newline\newline
Messages formats are defined in \textbg{interface files} which usually constitute entire packages.
\newline\newline
This lecture is \href{https://github.com/robmasocco/DAFN23_Robotics_3}{\color{blue}\underline{here}}.
\end{frame}
